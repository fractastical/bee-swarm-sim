Bee Memory
Dance memories encode angle + distance with drift. Flower memories decay unless reinforced by successful trips.
Bee Behavior
Scout → forage → return → dance. Idle hive bees copy dances only when physically close enough.
Modes
Default balanced recruitment. Try raising noise and lowering scent to watch the colony lose coherence.
Core primitives
Primitive glossary (definitions + citation years):
  • Waggle: positive recruitment signal; dancers advertise food direction + distance.
  • Stop: inhibitory signal that reduces/interrupts ongoing recruitment pressure.
  • Tremble: unloading-stress signal that recruits processing labor inside the hive.
  • Shaking: broad activation signal that mobilizes idle workers.
